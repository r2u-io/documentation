# Real2U Integration

Real2U Integration SKU

## Abrir AR
```
openAR({sku, name})
```
### Exemplo
```
openAR({42198124, Puff Bonjour})
```

## Criar visualizador 3D
```
create3DViewer({sku, name})
```
### Exemplo
```
create3DViewer({42198124, Puff Bonjour})
```
